# Hi, I'm Joel Kaplan 👋

Computer Engineering student at the Technion, focused on digital design, ASICs, FPGAs, embedded systems, and hardware/software co-design.

I enjoy taking hardware from RTL through verification, FPGA prototyping, and physical implementation. My recent work includes two hardened SKY130 Tiny Tapeout game designs and a SystemVerilog VGA arcade game running on a DE10-Standard FPGA.

## Featured projects

| Project | Highlights |
| --- | --- |
| [JA Achtung Full — 1x2 Tiny Tapeout design](https://github.com/venusjoel/JA-Achtung-Full) | A two-player hardware game with 64-direction movement, a 2-bit QSPI PSRAM framebuffer, dual gamepads, and direct 640×480 VGA. Verified against Python reference models through RTL, gate-level, and RTL-to-GDS checks. |
| [FPGA VGA Arcade Game](https://github.com/venusjoel/FPGA-VGA-Game) | A Pokémon-inspired arcade game implemented in SystemVerilog for the Terasic DE10-Standard, including hardware sprite composition, collision detection, PS/2 input, and audio. |
| [JA Achtung Compact — 1x1 Tiny Tapeout design](https://github.com/venusjoel/JA-Achtung-Compact) | An area-optimized one-tile version using a 1-bit QSPI PSRAM framebuffer, direct VGA, and the same hardware/software co-verification approach. |
| [WeStream](https://github.com/venusjoel/WeStream) | A Zoom-style screen, voice, and text streaming platform built from scratch in Python over raw TCP sockets as my 2020 high-school final project. |
